# supply_chain_project
Operation analysis project of Dataco supply chain dataset using SQL + Power BI
## Project Overview
Built an end-to-end analytics project using MySQL and Power BI to analyze supply chain performance, profitability, customer segments, and delivery delays.

## Tools Used
- MySQL
- DBeaver
- Power BI
- CSV Dataset

## Dataset Size
- 92,737 rows
- 44,449 unique orders

## Project Workflow
1. Imported raw CSV into MySQL
2. Cleaned column names and date fields
3. Validated null values and duplicates
4. Created KPI analysis using SQL
5. Analyzed region profit, shipping delays, customer segments
6. Built executive Power BI dashboard

## Key KPIs
- Revenue: $18.19M
- Profit: $1.97M
- Profit Margin: 10.84%
- Avg Order Value: $409.25

## Key Insights
- First Class shipping had highest late delivery rate (97%)
- Western Europe highest profitable region
- Consumer segment generated highest revenue share
- 17K+ rows had negative profit
