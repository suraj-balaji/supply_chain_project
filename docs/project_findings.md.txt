**Key findings:
KPI analysis
1. Late deliveries are unusually high at 56% of rows/orders.
2. Total profit margin is ~10.8%.
3. Average order value is $409.
4. Total order are 44k.
5. late deliveries usually high in standard class shipment mode.

6. First Class shipping has 97% late delivery rate.
7. Delay issue spans multiple regions.
8. Premium shipping process requires immediate review.

Operation analysis
1. First Class promises 1 day shipping but averages shipping days 2.
2. Second Class promises 2 days shipping but averages shipping days ~4.
3. Standard Class performs near expectation.
4. Premium shipping products drive most late-delivery complaints.

Profitability analysis
1. Western Europe and Central America are strongest profit regions.
2. 18.6% of transaction rows are loss-making.
3. Standard Class generates highest total profit.
4. First Class has highest average profit per row.
5. Higher discounts consistently reduce margin.

Product profitability Analysis
1. Fitness and outdoor products dominate total profit.
2. Dell Laptop has low volume but very high profit per row.
3. Several niche products are loss-making.
4. Product mix optimization could improve margin.

customer segment analysis
1. Consumer segment contributes highest total revenue.
2. Corporate segment nearly matches Consumer profit with fewer customers.
3. Corporate customers appear more valuable per customer.
4. Several repeat/VIP customers show strong revenue concentration.



**Recommended Actions
KPI analysis
1. Investigate worst-performing regions.
2. Review shipping mode SLAs and carrier performance.

3. Audit promised vs actual delivery SLA by shipping mode.
4. Validate delivery status definitions in source system.

Operation analysis
1. Recalibrate shipping SLAs for First/Second Class.
2. Investigate premium carrier fulfillment process.
3. Promote Standard Class if performance more reliable.

Profitability analysis
1. Review pricing strategy for 20%+ discounts.
2. Audit loss-making products/orders.
3. Expand profitable regions strategically.
4. Improve weak-margin geographies.

Product profitability Analysis
1. Increase focus on top profitable SKUs.
2. Review pricing of negative-profit products.
3. Protect inventory for high-margin items.

customer segment analysis
1. Increase Corporate acquisition campaigns.
2. Build loyalty programs for top repeat customers.
3. Segment dashboards by customer type monthly.
